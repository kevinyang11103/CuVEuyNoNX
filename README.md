# 基于Python的期货程序化交易系统 python381

## 项目概述

本项目是一个基于Python的期货程序化交易系统，使用了Flask作为后端Web框架，搭配MySQL数据库进行数据存储，前端采用Vue.js框架进行构建。

## 技术栈

- Python 3.8
- Flask
- MySQL 5.7
- Vue.js

## 功能模块

系统包含但不限于以下功能模块：

- 用户管理
- 交易策略配置
- 历史数据分析
- 实时行情展示
- 程序化交易执行

## 系统角色

- 管理员：负责用户管理和系统维护
- 用户：可以进行交易策略配置和执行交易操作

## 运行环境

- Python 3.8及以上版本
- Flask框架正常运行依赖的环境
- MySQL 5.7数据库环境
- Vue.js环境，用于前端页面展示

## 部署步骤

1. 安装Python 3.8环境
2. 安装Flask框架及相关依赖
3. 配置MySQL 5.7数据库
4. 部署Vue.js前端项目
5. 运行后端Flask应用
